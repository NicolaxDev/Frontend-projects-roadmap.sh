
# 📄 Single-Page CV

## 🎯 Description

This project consists of creating a **one-page resume** using only **pure HTML**.  
It is part of the list of projects proposed by [roadmap.sh](https://roadmap.sh/projects/single-page-cv).

The main objective is to practice the basics of HTML and content structuring without relying on CSS or JavaScript.

---

## 🛠️ Technologies used
- **HTML5**
